# Git Practice Playground

A small static website for practicing common Git commands inside the browser.

## Files
- `index.html`
- `styles.css`
- `script.js`

## Usage
1. Open `index.html` in your browser.
2. Read the prompt and type the appropriate `git` command.
3. Click `Check` or press Enter.
4. Click `Next Challenge` to try other scenarios.

## Customization
- Add more prompts in `script.js` under the `challenges` array.
- Adjust styles in `styles.css`.
